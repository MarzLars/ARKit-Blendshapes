# ARKit Blendshapes
Resources: 
* https://developer.apple.com/documentation/arkit/arfaceanchor/blendshapes
* https://arkit-face-blendshapes.com/
  
### ARKit Blendshapes List
Here is the complete list of ARKit blendshapes used for facial animation:

#### Non-duplicated blendshapes

##### Eyebrows
  - browInnerUp

##### Eyes
  - eyeLookUp
  - eyeLookDown
  - eyeLookIn
  - eyeLookOut

##### Cheeks & Nose
  - cheekPuff

##### Jaw
  - jawOpen
  - jawForward
  - jawLeft
  - jawRight

##### Mouth
  - mouthFunnel
  - mouthPucker
  - mouthRollUpper
  - mouthRollLower
  - mouthShrugUpper
  - mouthShrugLower
  - mouthClose

##### Tongue
  - tongueOut
